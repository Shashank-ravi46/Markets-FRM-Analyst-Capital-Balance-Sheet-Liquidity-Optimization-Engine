## What This Project Does
One paragraph. Plain English. No jargon.
"This system calculates regulatory capital requirements for a 
£3.4B UK equity trading book (BP, Shell, HSBC, Barclays, 
Lloyds, BAE Systems, IAG, JPMorgan) using the FRTB 
Standardised Approach, runs CCAR stress tests across three 
macro scenarios, and optimises desk-level RWA allocation 
to maximise Return on Regulatory Capital."

## Why It Matters (Business Context)
- FRTB goes live across UK banks in 2025 — every major 
  trading desk needs this exact calculation
- CCAR stress tests are mandatory for G-SIBs annually
- FRM teams at JPM, Goldman, Barclays run these models daily
